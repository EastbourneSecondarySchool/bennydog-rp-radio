## Quick Setup

1. Clone your repo:
   ```
   git clone https://github.com/EastbourneSecondarySchool/walkie-talkie-system.git
   cd walkie-talkie-system
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the site locally:
   ```
   npm start
   ```
4. Deploy to GitHub Pages:
   ```
   npm run deploy
   ```
